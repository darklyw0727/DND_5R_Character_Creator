# D&D 5e（XPHB）繁體中文術語翻譯技能

## 技能用途

本技能提供 D&D 5e（2024版不全書 XPHB）的台灣繁體中文官方術語對照，適用於：

- 翻譯 5etools JSON 檔案（如 `spells-xphb.json`）
- 提供法術、職業、技能、規則術語的正確中文譯名
- 確保譯名符合台灣繁體中文版官方用語

**參考來源**：台灣繁體中文版不全書（XPHB）官方術語 ＋ [5e CHM 繁體中文版](https://5echm.kagangtuya.top/)

---

## 核心翻譯原則

1. **保持 JSON 格式**：翻譯時不改變 JSON 結構與鍵值（key）
2. **專有名詞優先對照本技能術語表**：表內有的用表內術語，沒有的保留原文
3. **`{@tag}` 標記處理**：翻譯 5etools tag 中的顯示文字，保留骰子記號（如 `3d6`）
4. `original_name` 欄位可加入原文名稱供參照
